# CompanyBrain dashboard

Next.js 15 App Router dashboard for CompanyBrain.

```sh
pnpm --filter @companybrain/web dev   # http://localhost:3000
```

Talks to the API at `NEXT_PUBLIC_API_URL` (default http://localhost:3333) with cookie auth.
Design: amber phosphor on near-black, one accent, dense and fast. Inter + JetBrains Mono.
